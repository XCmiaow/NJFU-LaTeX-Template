# NJFU-LaTeX-Template v2.1.0 Release Notes Draft

这是南京林业大学课程论文 LaTeX 模板的 v2.1.0 版本。

## 普通同学下载哪个文件

如果你只是想写课程论文，请下载：

- `njfu-course-paper-v2.1.0-overleaf.zip`

上传到 Overleaf 后，在 **Menu -> Compiler** 中选择 **XeLaTeX**。

## 这个版本更新了什么

- 新增 [模板文件地图](template-map.md)，说明每个写作任务应该修改哪个文件。
- 新增 [写作配方](writing-recipes.md)，提供章节、引用、图片、表格、公式、代码和盲审的最小示例。
- 扩展 Overleaf 包内 `README.md`，普通同学只下载 zip 也能看到常用写作方法。
- 增强故障排查说明，补充参考文献、图片文件名和写作配方入口。
- 扩展新手文档检查，避免后续改动漏掉文件地图和写作配方。

## 不兼容变更

本版本没有不兼容变更。v2.0.0 用户可以直接下载新版 Overleaf 包继续使用。

## 发布资产

- `njfu-course-paper-v2.1.0-overleaf.zip`：Overleaf 模板包，普通用户优先下载。
- `njfu-course-paper-v2.1.0.pdf`：课程论文模板预览 PDF。
- `NJFU-LaTeX-Template-v2.1.0.zip`：完整源码包，适合本地使用或二次开发。
- `main-v2.1.0.pdf`：完整示例 PDF。

## 使用前请注意

- 本项目不是学校官方模板。
- 本项目适合课程论文、课程作业、课程设计说明文档等轻量场景。
- 毕业论文、学位论文、开题报告请以学院或学校官方要求为准。
