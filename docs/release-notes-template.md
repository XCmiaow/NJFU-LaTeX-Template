# Release Notes Template

发布 GitHub Release 时，可以复制下面内容作为说明。

```markdown
## NJFU-LaTeX-Template vX.Y.Z

这是南京林业大学课程论文 LaTeX 模板的 vX.Y.Z 版本。

### 普通同学下载哪个文件

如果你只是想写课程论文，请下载：

- `njfu-course-paper-vX.Y.Z-overleaf.zip`

上传到 Overleaf 后，在 **Menu -> Compiler** 中选择 **XeLaTeX**。

### 这个版本更新了什么

- 更新点 1。
- 更新点 2。
- 更新点 3。

### 不兼容变更

- 如果这是 major 版本，在这里说明旧版本用户需要怎么迁移。
- 如果没有不兼容变更，删除本节。

### 发布资产

- `njfu-course-paper-vX.Y.Z-overleaf.zip`：Overleaf 模板包，普通用户优先下载。
- `njfu-course-paper-vX.Y.Z.pdf`：课程论文模板预览 PDF。
- `NJFU-LaTeX-Template-vX.Y.Z.zip`：完整源码包，适合本地使用或二次开发。
- `main-vX.Y.Z.pdf`：完整示例 PDF。

### 使用前请注意

- 本项目不是学校官方模板。
- 本项目适合课程论文、课程作业、课程设计说明文档等轻量场景。
- 毕业论文、学位论文、开题报告请以学院或学校官方要求为准。
```
