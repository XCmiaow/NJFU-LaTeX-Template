# NJFU-LaTeX-Template v2.2.0 Release Notes Draft

这是南京林业大学课程论文 LaTeX 模板的 v2.2.0 版本。

## 普通同学下载哪个文件

如果你只是想写课程论文，请下载：

- `njfu-course-paper-v2.2.0-overleaf.zip`

上传到 Overleaf 后，在 **Menu -> Compiler** 中选择 **XeLaTeX**。

## 这个版本更新了什么

- 新增 [Overleaf 手把手使用教程](overleaf-step-by-step.md)，从下载 Release 到上传 Overleaf、设置 XeLaTeX、编译和下载 PDF 都逐步说明。
- 新增 [第一篇课程论文完整演示](first-paper-walkthrough.md)，用一个假题目演示如何填写基本信息、摘要、正文、图片和参考文献。
- 重写 [第一次使用完整流程](student-quickstart.md)，把原来的速查说明扩展成完整入门流程。
- 扩展 Overleaf 包内 `README.md`，普通同学只下载 zip 也能照着完成第一篇课程论文。
- 扩展新手文档检查，确保这些教程进入后续发布包。

## 不兼容变更

本版本没有不兼容变更。v2.0.0 和 v2.1.0 用户可以直接下载新版 Overleaf 包继续使用。

## 发布资产

- `njfu-course-paper-v2.2.0-overleaf.zip`：Overleaf 模板包，普通用户优先下载。
- `njfu-course-paper-v2.2.0.pdf`：课程论文模板预览 PDF。
- `NJFU-LaTeX-Template-v2.2.0.zip`：完整源码包，适合本地使用或二次开发。
- `main-v2.2.0.pdf`：完整示例 PDF。

## 使用前请注意

- 本项目不是学校官方模板。
- 本项目适合课程论文、课程作业、课程设计说明文档等轻量场景。
- 毕业论文、学位论文、开题报告请以学院或学校官方要求为准。
