# Format Scope

本项目是南京林业大学课程论文 LaTeX 模板，不是学校官方模板。

## 适合使用

- 课程作业。
- 课程结课论文。
- 课程设计说明文档。
- 需要封面、目录、摘要、正文、参考文献的轻量论文。

## 不建议直接使用

- 毕业论文。
- 学位论文。
- 开题报告。
- 学院或课程已经发布了专用 Word/LaTeX 模板的正式材料。

这些场景通常有更严格的格式要求，提交前应以老师、学院或学校发布的文件为准。

## 当前覆盖的格式

- A4 页面和中文课程论文正文排版。
- 课程论文封面字段：题目、课程、学院、专业、学号、姓名、授课教师。
- 目录。
- 中文摘要和关键词。
- 可选英文摘要。
- 正文章节标题。
- 图、表、公式、算法、代码示例。
- BibTeX 参考文献。
- 盲审模式：隐藏姓名、学号、授课教师。

## 当前不保证的格式

- 学校官方毕业论文规范。
- 不同学院的专门封面。
- 不同老师对行距、字号、标题编号的个性化要求。
- GB/T 7714 等特定参考文献样式。
- PDF/A、查重系统专用格式或归档格式。

## 提交前检查

提交给老师前，至少检查：

- 封面信息是否正确。
- 目录页码是否正常。
- 摘要和关键词是否完整。
- 正文第一页是否符合老师要求。
- 图表标题和编号是否正常。
- 参考文献是否显示。

更完整的逐项检查见 [提交前格式检查](format-checklist.md)。
